== Description ==

Clever Fox plugin to enhance the functionality of free themes made by Nayra Themes. More than 50000+ trusted websites with Nayra Themes. It provides intuitive features to your website. 20+ Themes compatible with Clever Fox. See below free themes listed here. Avril is one of Popular themes in our collections.

= The Clever Fox plugin features include: =

- Import Pre-made Section in 1 Click
- Beautiful Sections on Homepage
- Edit content via Live Customizer
- Manage Font Typography
- Background Animations
- Kenburn Effect on Slider
- Compatible with WPML & Polylang
- Translation Ready
- Responsive for All Devices
- Cross Browser Compatibility
- and More Features

== Installation ==

= Option 1. =

- Upload the entire `clever-fox` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Clever Fox" —> Click install.
- After plugin is downloaded, click active plugin
